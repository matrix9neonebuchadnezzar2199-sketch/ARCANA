# ARCANA Blender Addon

> Coming in Phase 3

This addon will connect Blender to the ARCANA MCP server via WebSocket,
enabling natural language control of Blender operations.

## Planned Features

- Mesh creation and editing
- Modifier management
- Material and texture control
- Animation and keyframe tools
- VRM/VRChat avatar tools
- Sculpt mode operations
- Render settings (Cycles/Eevee)
- UV unwrapping

## Requirements (planned)

- Blender 3.6+
- Python 3.10+
- ARCANA MCP server running
